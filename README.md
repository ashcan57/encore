# Encore Repository

Welcome to the **Encore Repository**, home of the **Encore Build Wizard** for Kodi.

---

### 📦 Installation Instructions

1. **Open Kodi**
2. Go to **Settings → File Manager → Add Source**
3. Select **<None>** and enter the URL:


4. Name the source **Encore** and press **OK**.
5. Go back to **Add-ons → Install from zip file → Encore**.
6. Install the repository: **repository.encore.zip**.
7. Then go to **Install from repository → Encore Repository → Program add-ons → Encore Build Wizard**.

---

### 🧠 What It Does
The **Encore Build Wizard** downloads the official Encore build ZIP to Kodi’s `special://home/addons/packages/` directory.

*(It does not extract or overwrite your Kodi setup; it’s designed for safe personal use.)*
